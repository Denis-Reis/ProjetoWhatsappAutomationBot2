# Projeto WhatsApp Automation Bot

Bot de automação para WhatsApp usando **Flask** e **WhatsApp Cloud API**.

## 🚀 Instalação

Clone o repositório:
```bash

git clone https://github.com/Denis-Reis/ProjetoWhatsappAutomationBot2.git
cd ProjetoWhatsappAutomationBot2

python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt

📲 Uso
Envie mensagens para o número de teste do WhatsApp.

O bot recebe via webhook, processa e responde automaticamente.

🛠️ Tecnologias
Python 3.13

Flask

Requests

WhatsApp Cloud API
